# fasten-timer
Minimalistischer Fastentimer als Progressive Web App (PWA). Verfolge deine Fastenphasen, passe die Startzeit nachträglich an und nutze den Timer lokal im Browser. Kein Login, keine Cloud – deine Daten bleiben erhalten, bis du stoppst. Vibe-Coding-Projekt mit Google Gemini.
